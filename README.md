# GoodLauncher 제작기

OpenLauncher를 기반으로 주제를 정하여 테마를 적용시켜 볼 것 입니다.


## 테마 (추가 및 수정 가능)

    *공부
    *게임

### 기능 (추가예정)

  * 공부
    * 굿런처 전용 캘린더 
    
  * 게임
   * 프레임 기능 추가
    
### 수정된 기능
   * 한글패치
   * GoodLauncher Icon 변경
   * 테마 변경
   * AppDrawer에 SearchBar 추가
   * GoodLauncher Calendar 추가
   * 게임모드에 프레임 출력기능 추가
